# Week 2 - Day 1 SQL Queries

## Introduction

This folder contains SQL practice queries based on the College Management System (CMS) database. The queries are designed to practice basic SQL operations and improve understanding of database concepts.

## Topics Covered

* SELECT Statements
* CONCAT Function
* ORDER BY Clause
* INNER JOIN
* WHERE Condition
* Table Aliases

## Questions and Description

### Question 1

Display all records from the `Student` table.

### Question 2

Display the address and city together in a single column using the `CONCAT` function.

### Question 3

Display the address and city in descending order.

### Question 4

Display the address and city in ascending order.

### Question 5

Display student names along with their department names for students belonging to Coimbatore city using `INNER JOIN`.

### Question 6

Display department names whose block number is 3.

College Management System (CMS)

## SQL Concepts Practiced

* Retrieving data using `SELECT`
* Combining columns using `CONCAT`
* Sorting records using `ORDER BY`
* Joining tables using `INNER JOIN`
* Filtering records using `WHERE`
* Using table aliases for better readability

## Learning Outcome

After completing these queries, you will be able to:

* Write basic SQL queries
* Retrieve and filter records from tables
* Sort data in ascending and descending order
* Combine multiple columns into a single output
* Perform joins between related tables
* Improve query readability using aliases
